# /implement

选择一个变更开始执行。

## 用法

```bash
npx tsx commands/implement.ts [change-name]
npx tsx commands/implement.ts my-feature --no-worktree
```

## 执行

⚠️ **Worktree 将自动创建 (除非使用 --no-worktree)**

请立即调用 `superpowers:executing-plans` skill 开始工作。

**REQUIRED SUB-SKILL:** Use superpowers:executing-plans
