# Tech Stack

## Core
- **Runtime**: Node.js (>=18.0.0)
- **Language**: TypeScript (^5.4.5)
- **Module System**: ESM (type: module)

## Development & Build
- **Build Tool**: pure `tsc`
- **Linter**: ESLint
- **Testing**: Vitest (^1.5.0)

## Libraries
- **CLI Framework**: Custom (using `superpowers-fusion` bin entry)
- **Utilities**: `glob`, `minimatch`, `dotenv`, `zod`, `yaml`, `@ltd/j-toml`
- **Compiler**: Babel (parser/traverse/types) for AST manipulation
